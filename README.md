# Презентация для партнеров obanket.ru

Современная HTML-презентация для привлечения партнеров на платформу obanket.ru.

## О проекте

Этот проект представляет собой одностраничный сайт-презентацию, предназначенную для демонстрации преимуществ партнерства с obanket.ru ресторанам, отелям, барам и другим заведениям.

## Функциональность

- Адаптивный дизайн для всех устройств
- Плавная прокрутка к разделам
- Анимации при скролле
- Интерактивные элементы
- Активные контактные ссылки

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- JavaScript (нативный)
- Font Awesome icons
- Unsplash Images
